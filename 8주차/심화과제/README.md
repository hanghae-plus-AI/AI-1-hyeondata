    - 다음 두 가지 방향 중 하나로 LLM 서비스를 경량화하시면 됩니다:

        2. **큰 LLM 활용:** 같은 dataset으로 학습됐지만 더 큰 open LLM을 활용할 수 있도록 경량화하시면 됩니다. 예를 들어, `gemma-2b-it`이라는 모델로 LLM 서비스를 개발했지만 경량화를 진행하여 `gemma-9b-it`로 LLM 서비스를 구현하게끔 할 수 있습니다.
